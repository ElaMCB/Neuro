# Neuro Scripts

Utility scripts that demonstrate Neuro's capabilities and provide helpful tools.

## Job Search Scripts

Located in `job_search/`:

- **`improved_job_search.py`** - Main job search engine (recommended)
- **`neuro_job_search.py`** - Neuro integration for job searching
- **`resume_preparer.py`** - Resume optimization and preparation
- **`display_results.py`** - Display and format job search results
- **`generate_html_report.py`** - Generate beautiful HTML reports
- **`schedule_job_search.py`** - Schedule automatic job searches
- **`send_job_search_email.py`** - Email job search results

### Usage

From the Neuro root directory:

```bash
# Run job search from a .neuro file
neuro examples/my_job_search.neuro

# Or run Python scripts directly
python scripts/job_search/improved_job_search.py
```

## Legacy Scripts

Some older implementations are kept for reference:
- `advanced_job_search.py` - Earlier version
- `better_job_search.py` - Earlier version

For new projects, use `improved_job_search.py`.
